# JointTheMove

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.18.

## Initialiser le projet

```bash
cd JoinTheMove
npm i
```

### Lancer le backend :

```bash
cd backend
npx json-server db.json
```

### Lancer le frontend :

```bash
npm run start
```
